# Cognito JobFit AI Analytics 🔮
### *Enterprise Profile Alignment & Semantic Screening Engine*

Cognito JobFit AI is a high-performance talent screening dashboard engineered to analyze profiles against corporate job requirements using advanced Large Language Models (LLMs). Built with a custom neon executive theme, the system parses data, extracts core competencies, measures alignment thresholds, and surfaces critical skill gaps in real-time.

---

## 🚀 Key Architectural Features
* **Semantic Resonating Match Engine:** Bypasses legacy keyword matching using `llama-3.3-70b-versatile` to evaluate conceptual profile alignment.
* **Automated Ingestion Pipeline:** High-speed raw text parsing from multi-page PDFs utilizing `pypdf`.
* **Strategic Vector Insights:** Generates deterministic structural summaries, explicit competency indicators, and gap readouts.
* **Premium Interactive Interface:** A high-contrast, cyberpunk-inspired dark workspace engineered entirely inside `Streamlit`.
* **Dynamic Neural Scanner Component:** Features custom-embedded CSS laser scanning animations to visualize processing sequences.

---

## 🛠️ Tech Stack & Dependencies
* **Core Framework:** Streamlit
* **Orchestration:** LangChain Core / LangChain Groq
* **Inference Provider:** Groq Cloud API
* **Base Models:** Llama 3.3 70B Versatile (`llama-3.3-70b-versatile`)
* **Document Parsers:** PyPDF / IO Stream Matrix
